4. Fixtures & Group Tables

Enter the scores for each group match. The date/time shown is UK time. The website automatically calculates wins, draws, losses, goals, goal difference and points.

Group Stage Scoring

Win = 3 points, Draw = 1 point, Loss = 0 points. Leaderboard group-stage scores are then multiplied by pot: Pot 1 ×1, Pot 2 ×1.5, Pot 3 ×2. Tables are sorted by points, goal difference, goals scored, then team name.

Fixture results now update the leaderboard automatically using the agreed scoring system and pot multipliers.

Progress Bonuses

Select every team that reaches each stage. The site will automatically add the right bonus points using the pot multipliers. For example, if a Pot 3 team reaches the Quarter Final, it gets 15 × 2 = 30 bonus points.
